SimpleTimer allows you to create programmable timers / reminders. The latest release and source code can be found at http://cubelogic.org/simpletimer. Please send bug reports, questions and comments to Ettore Pasquini at support@cubelogic.org.

Support

SimpleTimer 1.2 was developed using the Cocoa/OpenStep framework. It was tested on Mac OS X 10.4 only, using a PPC processor: however, it should work fine on Mac OS X 10.3 and 10.2, as well as Intel machines. (SimpleTimer 1.2 is a universal binary.) You are welcome to report bugs.
